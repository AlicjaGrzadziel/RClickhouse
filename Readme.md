# Clckhs (Prototype)

**This is my first R-package, so be warned!**

It is a basic dplyr SQL-backend for the clickhouse database. The DBI interface is based on Hannes MÃ¼hleisen's [clickhouse-r package](https://github.com/hannesmuehleisen/clickhouse-r).

## Dev Info
 * Build and Reload Package:  `Ctrl + Shift + B`
 * Check Package:             `Ctrl + Shift + E`
 * Test Package:              `Ctrl + Shift + T`

Generate Namespace: `devtools::document()`
To use latest dplyr-package: ` devtools::install_github("hadley/dplyr")`
